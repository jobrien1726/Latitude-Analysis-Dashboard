Summary: Latitude vs. X

The goal of this project was to analyze how weather patterns change as we move closer to the equator. In order to complete this anaylsis, we extracted data from the OpenWeatherMap API and created a dataset where we observed different aspects of the weather from a random sample of over 500 different cities.

After putting together a dataset, we used MatPlotLib to plot the various weather factors against latitude. These factors included: Maximum Temperature, Humidity, Cloudiness and Wind Speed. This site provides access to the data collected, the subsequent visualizations created, as well as a summary of any patterns or correlations observed for each.
